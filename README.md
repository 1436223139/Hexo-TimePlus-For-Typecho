# TimePlus For Typecho相册模板

原作者两年都没有维护这个主题了，很喜欢这个主题想要更快速的迭代优化，所以新开了这个坑。基于原作者的[Time主题](github.com/wclk/time)更新全新的TimePlus。

主题基于原Time主题可能有个人设计偏好，如果你不喜欢这些改动，可以尝试原主题。
